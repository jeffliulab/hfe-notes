# 医疗器械

本分区将 ISO 14971、医疗器械 use error、URRA 实践与 EpiPen 工作簿整合成一套面向医疗器械风险管理的知识链路。

## 主题页面

- [ISO 14971 与医疗器械风险管理](iso_14971.md): 本页围绕 ISO 14971 标准文本与教学材料，整理医疗器械风险管理的基本流程、术语与与人因工作的接口。
- [医疗器械中的 URRA](medical_device_urra.md): 本页聚焦医疗器械场景下的 URRA，把 use scenario、critical task、harm 路径与控制策略放到监管语境中理解。
- [医疗器械中的使用错误](medical_device_use_errors.md): 本页聚焦医疗器械中的 use error、异常使用与界面诱发风险，强调错误并非只来自操作者，而是来自整个使用系统。
- [EpiPen URRA 工作簿](epipen_workbook.md): 本页用 EpiPen 表格示例把任务分析、潜在 use error、hazard、harm、severity、critical task 与 mitigation 映射成可执行的工作底稿。
